# Automation Scripts

This folder contains several DevOps-friendly automation tools written in Python.

Each tool demonstrates:
- CLI argument parsing
- File I/O
- Logging
- Error handling
- Real-world DevOps problem-solving

Included tools:

1. **File Scanner**
   - Scans directories
   - Identifies large files
   - Generates JSON reports
   - Great for log cleanup and build artifact management

More tools will be added later.
